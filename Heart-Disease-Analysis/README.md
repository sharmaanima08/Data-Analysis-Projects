# Heart Disease Risk Analysis

## Objective
To analyze key health indicators influencing heart disease risk.

## Dataset
Medical dataset containing age, cholesterol, blood pressure, and heart disease indicators.

## Tools Used
- Python
- pandas
- matplotlib
- seaborn

## Key Insights
- Age and cholesterol show strong correlation with heart disease
- High resting blood pressure increases risk

## Conclusion
This analysis helps identify major health risk factors useful for early diagnosis.
